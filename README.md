# joaoportolan.me

Portfólio pessoal de João Portolan — Desenvolvedor Frontend & UI/UX Designer.

## Stack

- React 18 + Vite
- Tailwind CSS (paleta golf pastel)
- Framer Motion
- Lucide React

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

O arquivo `CNAME` é copiado automaticamente para `dist/` pelo workflow de deploy.

## Deploy

O deploy é feito automaticamente via GitHub Actions ao fazer push na branch `main`.
